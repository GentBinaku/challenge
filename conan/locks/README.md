# Conan Lock Files

This directory stores `conan.lock` files generated by `conan install` commands.

Lock files capture the exact resolved versions of all dependencies, ensuring reproducible builds across different machines and times.

## Usage

Lock files are generated automatically:
```bash
conan install . --profile=conan/profiles/linux/gcc --lockfile-out=conan/locks/linux-gcc.lock
```

Or load an existing lock:
```bash
conan install . --profile=conan/profiles/linux/gcc --lockfile=conan/locks/linux-gcc.lock
```

Commit lock files to version control for deterministic CI/CD builds.
